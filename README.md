# Yandex Practicum Data Science Projects

This repository contains projects completed during the **Yandex Practicum Data Science Bootcamp**.

The projects demonstrate practical experience in data preprocessing, exploratory data analysis, feature engineering, and machine learning using real-world datasets.

---

## Skills and Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Data preprocessing
- Exploratory data analysis (EDA)
- Feature engineering
- Statistical analysis
- Machine learning

---

## Projects

1. **01_data_preprocessing**
   - Data preprocessing and data cleaning

2. **02_real_estate_analysis**
   - Apartment listings analysis and price factor research

3. **03_scooter_rental_analysis**
   - Business analysis of scooter rental service

4. **04_game_success_analysis**
   - Analysis of video game market and success factors

5. **05_customer_churn_prediction**
   - Bank customer churn classification

6. **07_oil_well_location_optimization**
   - Oil well location selection based on profit and risk

7. **08_gold_recovery_optimization**
   - Industrial ML model for gold recovery prediction

8. **09_data_privacy_transformation**
   - Data protection via feature transformation

9. **10_used_car_price_model**
   - Used car price prediction

10. **11_industrial_process_optimization_ml**
    - Steel temperature prediction and industrial optimization

---

## Notes

- The first two projects (**01–02**) are **fully translated into English**.
- In the remaining projects, only the following parts are translated:
  - project title  
  - project description and objective  
  - research plan (at the beginning)  
  - final conclusions and results (at the end)

- Datasets are **not included in the repository**.  
  Each project contains a function at the beginning that downloads the dataset directly from a Yandex Cloud link.  
  These links are publicly accessible, so all data can be easily reproduced.

---

These projects were originally completed in Russian as part of the course requirements.  
English translations were added to improve accessibility for international readers.
